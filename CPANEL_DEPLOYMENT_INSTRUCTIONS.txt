- Admin panel for content management (uses localStorage)

🔧 Deployment Steps:

1. EXTRACT FILES
   - Download caretheplanet-cpanel-ready.zip
   - Extract ALL files directly (no subfolder needed)
   - You should see: index.html, _next folder, admin folder, etc.
   - ⚠️ IMPORTANT: Extract files directly, not into a subfolder!

2. UPLOAD TO CPANEL
   - Login to your cPanel File Manager
   - Navigate to public_html directory
   - Upload ALL extracted files DIRECTLY to public_html root
   - Make sure index.html is directly in public_html (not in any subfolder)
   - Structure should be:
     public_html/
     ├── index.html
     ├── zenopay-proxy.php  ← IMPORTANT: PHP file for payments
     ├── _next/
     ├── admin/
     ├── blog/
     ├── contact/
     └── ...

3. SET FILE PERMISSIONS
   - Directories: 755
   - Files: 644
   - zenopay-proxy.php: 644 (PHP file for payment processing)
   - Most cPanel hosting sets these automatically

4. VERIFY PHP SUPPORT
   - Your hosting must support PHP (most cPanel hosting does)
   - PHP 7.4+ recommended
   - cURL extension must be enabled (standard on most hosts)

4. TEST YOUR WEBSITE

   - Test payment with phone: 0754546567 (test mode)
   - Admin panel: https://yourdomain.com/admin/login

     Password: caretheplanet2024
