🔧 DEPLOYMENT STEPS:

1. 📁 UPLOAD FILES:
   - Login to your cPanel File Manager
   - Navigate to public_html directory
   - Upload ALL files from the 'out' folder DIRECTLY to public_html root
   - Make sure index.html is directly in public_html (not in any subfolder)
   - File structure should be:
     public_html/
     ├── index.html
     ├── zenopay-proxy.php  ← IMPORTANT: PHP file for payments
     ├── _next/
     ├── admin/
     ├── blog/
     ├── contact/
     └── ...

2. ⚙️ CONFIGURE PAYMENTS:
   - Edit zenopay-proxy.php file
   - Replace 'your_zenopay_api_key_here' with your actual ZenoPay API key
   - Save the file

3. 🔐 SET FILE PERMISSIONS:
   - Set zenopay-proxy.php permissions to 755
   - Ensure PHP files can execute

4. 🌐 VERIFY DEPLOYMENT:
   - Visit: https://yourdomain.com
   - Test navigation between pages
   - Try making a test payment with phone: 0754546567
